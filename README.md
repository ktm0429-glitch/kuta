# kuta - GOLD Trading EA Project

## 概要

このリポジトリは、MetaTrader5用のGOLD（XAUUSD）トレーディングシステムの開発プロジェクトです。

### プロジェクト構成

1. **GOLD SMC EA** - Smart Money Concept に基づく構造認識型自動売買システム
   - BoS (Break of Structure) 検出
   - Order Block (OB) 認識
   - Kicker パターン検出
   - 再テスト条件による精密なエントリー

2. **kutaGOLD Titan Series** - AI統合型進化システム
   - v1.x〜v3.5Φ: 段階的な機能拡張
   - 環境適応型AI (EnvAdapt, PhaseScore, AutoMode)
   - TitanFX実運用対応

## 重要な設定

### 言語設定
このプロジェクトでは、**すべてのAI応答は日本語で行われます**。

`.claude/instructions.md` に設定が記載されています。
